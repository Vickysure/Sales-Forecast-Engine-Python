#  Sales Forecasting Engine with Python

## Project Overview

This project is a beginner-friendly Python case study that simulates a simple sales forecasting and inventory management system for **BrightMart Retail Limited**.

The objective of the project is to analyze weekly sales performance for two retail products, generate sales forecasts, evaluate inventory levels, and identify products that require restocking or marketing attention.

This project demonstrates the application of Python fundamentals to solve practical business problems and serves as one of my first hands-on data analytics projects.

## Business Scenario

BrightMart Retail Limited wants to monitor the weekly performance of its products and make informed decisions based on sales trends.

Using Python, this project analyzes:

- Weekly revenue generated
- Sales target achievement
- Forecasted sales for the following week
- Inventory status
- Restocking requirements
- Product availability
- Business recommendations

## Products Analyzed

-  Wireless Mouse
-  Bluetooth Speaker

## Project Objectives

- Calculate weekly revenue for each product.
- Measure performance against weekly sales targets.
- Forecast expected sales growth.
- Evaluate current inventory levels.
- Determine whether products require restocking.
- Compare the overall performance of both products.
- Recommend which product should be prioritized in future marketing campaigns.
